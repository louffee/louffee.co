# Heading

The heading component is used to display a title or a section heading. It is 
used to give a clear indication of what the user is looking at. Also, it 
improves Search Engine Optimization (SEO) by providing a clear indication of 
what the page is about.

## Usage

```tsx
import Heading from '@louffee/ui/heading'

<Heading variant='h1'>Lorem ipsum</Heading>
```
