# Icon

<GitHubRepoPath path="packages/ui/src/components/icon/icon.tsx" />

The component which renders an icon based on the given name from our 
iconography. The icons are SVGs found in the internal modules of the Louffee's 
UI package.

All the icons are available in the [iconography](#?path=/docs/style-guide-iconography--docs)
page of the style guide.

## Usage

```tsx
import Icon from '@louffee/ui/icon'

<Icon name="GitHubLogo" size={60} className="text-black" />
```

Then, the result will be:

<Icon name="GitHubLogo" size={60} className="text-black" />
