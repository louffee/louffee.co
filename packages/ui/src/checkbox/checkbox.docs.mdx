# Checkbox

<GitHubRepoPathLink path="packages/ui/src/checkbox/checkbox.tsx" />

The checkbox component is used to select or deselect an option.

## Usage

```tsx
import Checkbox from '@louffee/ui/checkbox'

<Checkbox />
```

Then, the checkbox would appear in the app like this:

<div className='flex items-center'>
  <Checkbox />
</div>
