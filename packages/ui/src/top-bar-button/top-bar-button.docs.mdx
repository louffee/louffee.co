# TopBarButton

The component that renders an [anchor](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a)
HTML element that triggers a redirection via the Next.js 
[Link API](https://nextjs.org/docs/app/api-reference/components/link).

## Usage

```tsx
import TopBarButton from '@louffee/ui/top-bar-button'

<TopBarButton href="/?path=/docs/components-topbarbutton--docs">
  Lorem Ipsum
</TopBarButton>
```

Then, the component will render an anchor HTML element with the text "Lorem 
Ipsum" and a redirection to the home page.

<TopBarButton href="/?path=/docs/components-topbarbutton--docs">Lorem Ipsum</TopBarButton>
