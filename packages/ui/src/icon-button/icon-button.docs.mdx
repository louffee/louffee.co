# IconButton

<GitHubRepoPath path="packages/ui/src/icon-button/icon-button.tsx" />

The IconButton component is a simple IconButton that can be used to trigger an action.

## Usage

```tsx
import IconButton from '@louffee/ui/icon-button'

<IconButton variant="primary" icon="Rocket" />
```

Then, you can use the IconButton component in your app like this:

<IconButton variant="primary" icon="Rocket" />
