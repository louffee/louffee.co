# TopBar

The top bar is a container which renders at the top of the screen. It is used to
display the anchors which link to the different sections of the page or to
different pages.

For more information about the links, please take a look at the TopBarButton 
component. Internally, the TopBar utilises the **ViewportAxis** and the 
**AfloatBar** components to compose the final layout.

## Usage

```tsx
import TopBar from '@louffee/ui/top-bar'
import TopBarButton from '@louffee/ui/top-bar-button'

<TopBar>
  <TopBarButton href="#">Lorem</TopBarButton>
  <TopBarButton href="#">Ipsum</TopBarButton>
  <TopBarButton href="#">Sit</TopBarButton>
  <TopBarButton href="#">Dolor</TopBarButton>
</TopBar>
```
