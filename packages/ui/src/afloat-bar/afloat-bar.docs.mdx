# AfloatBar

<GitHubRepoPath path="packages/ui/src/afloat-bar/afloat-bar.tsx" />

The AfloatBar component is a floating bar that can be used to display a message,
a call to action, or any other content.

## Usage

```tsx
import AfloatBar from '@louffee/ui/afloat-bar'

<AfloatBar>Lorem Ipsum</AfloatBar>
```

Then, the component will be rendered as:

<AfloatBar>Lorem Ipsum</AfloatBar>

Adding actions to the AfloatBar component, such as the `Button` component, can 
be done as follows, producing the result below. Also, the `AfloatBar` component
receives the props from the native `div` HTML element, therefore, it can receive
custom classes and styles. See the following example:

```tsx
<AfloatBar className='flex justify-center items-center gap-3 p-3'>
  <Button variant="ghost">Lorem</Button>
  <Button variant="ghost">Ipsum</Button>
  <Button variant="ghost">Sit</Button>
  <Button variant="ghost">Dolor</Button>
</AfloatBar>
```
